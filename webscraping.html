Webscraping and Data Cleansing Top 100 Thai Dramas from 2000-2023

This project involved web scraping data on the top 100 Thai dramas from the year 2000-2023 from the website MyDramaList. The data was then cleaned using Python. The cleaned dataset can be used for further analysis to identify trends and patterns in Thai drama viewership over time.

Data Collection

The data was collected using the BeautifulSoup library in Python. The first step was to identify the HTML elements on the MyDramaList website that contained the relevant data. Once the elements were identified, the data was extracted and saved to a CSV file.

Data Cleaning

The data was then cleaned to remove any inconsistencies or errors. This involved removing duplicate entries, correcting spelling mistakes, and converting all data to a consistent format.
