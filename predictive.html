Logistic Regression Analysis of Online Purchase Intention

This project involved using Python to build a predictive model to predict online purchasing intention based on Google Analytics data from an e-commerce company. Logistic regression was used as the machine learning algorithm, as it is well-suited for binary classification tasks.

The model was trained on a dataset of historical customer data, which included features such as page views, bounce rate, and average session duration. The model was then evaluated on a held-out test set to assess its predictive performance.

The model achieved a high accuracy of 88% on the test set, demonstrating its ability to predict online purchasing intention with a high degree of accuracy. The model can be used by the e-commerce company to identify potential customers who are more likely to make a purchase, and to target these customers with personalized marketing campaigns.
