Cleaning data is a crucial step in working with data. Come explore this project as we clean a dataset of Data Science Jobs from Glassdoor. Webscraping with Python

Webscraping can provide an excellent way to extract data and build a dataset for further analysis. This project extracts data from a popular web platform of dramas filmed and produced in various Asian countries. Data Exploration with SQL

Data exploration provides insightful information about the data being explored. The PostgreSQL pgAdmin4 environment was used to explore the Coronavirus Deaths dataset provided by Our World of Data. Data Visualization with Tableau

Visualizations can provide a wonderful way to facilitate storytelling of the data to stakeholders. Within the visualization of the fictitious companies, we can see how a localized company lines up against a nationwide organization. These companies compare monthly charges based on internet service, add-on features, and customer churn status by gender. Data cleaning was completed via Python with the cleaned datasets imported into Tableau. Tableau was utilized for the interactive dashboard creation to glean insights and support decision-making.

Logistic Regression in Python

Logistic regression analysis allows for the prediction of a binary outcome. Utilizing this on data from online purchasing or ecommerce can predict whether or not a customer will purchase. Python was used to create this model with the use of Statsmodels’ Logit package to learn the relationship between the predictor attributes and outcome attribute. This was further visualized with Sci-Kit Learn’s confusion matrix’s plotting in Matplotlib.
